# cf_ai_chat

AI-powered chat application on Cloudflare (Workers AI, Pages, and optional state/workflow). Built for the Cloudflare Software Engineering Intern optional assignment.

**Status:** Build in progress. Running instructions and deployment link will be added when the app is complete.

## Planned components

- **LLM:** Llama 3.3 via Workers AI (or external LLM)
- **Workflow / coordination:** Workers (and optionally Workflows or Durable Objects)
- **User input:** Chat UI via Cloudflare Pages
- **Memory or state:** KV or Durable Objects for conversation history

Instructions to run locally (e.g. with Wrangler) and a deployed link will be added here once the application is built.
